# three_js

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```


### Start with docker
```
docker build -t vuejs-threejs .
docker run -it -p 8080:8080 --rm --name vuejs-threejs-app vuejs-threejs
```
